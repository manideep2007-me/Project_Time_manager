## ‚è∞ Time Tracking

### Starting Time Tracking

#### Method 1: Quick Start
1. Tap the **"+"** button on the Time Tracking tab
2. Select **Project** from the list
3. Select **Employee** from the list
4. Add optional **Description**
5. Tap **"Start Tracking"**

#### Method 2: From Project Details
1. Go to **Projects** tab
2. Tap on a project
3. Tap **"Start Time Tracking"**
4. Select employee and add description
5. Tap **"Start Tracking"**

### Stopping Time Tracking

#### Active Time Entries
1. Go to **Time Tracking** tab
2. Find the active time entry
3. Tap **"Stop"** button
4. Add optional **End Description**
5. Tap **"Stop Tracking"**

#### From Dashboard
1. View active entries on dashboard
2. Tap on the active entry
3. Tap **"Stop"** button

### Manual Time Entry

For timesheets or corrections:
1. Tap **"+"** button on Time Tracking tab
2. Select **"Add Time Entry"**
3. Fill in:
   - **Project**
   - **Employee**
   - **Start Time**
   - **End Time**
   - **Description**
4. Tap **"Save Entry"**
